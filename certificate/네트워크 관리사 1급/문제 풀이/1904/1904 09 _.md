# 문제 9
## SSH 프로토콜이 사용하는 포트 번호는?
	1. TCP 22번  
	2. TCP 23번
	3. UDP 24번
	4. UDP 25번

# 해설
## 주요 프로토콜 포트
- FTP-Data : 20
- FTP : 21
- SSH : 22
- Telnet : 23
- SMTP : 25
- DNS : 53
- HTTP : 80
- POP3 : 110
- NNTP : 119
- SNMP : 161
- HTTPS : 443
